
==========
localizaTE  
==========
 
de novo and homology based Transposable Element annotation system

Prerequesites: 

   - Repeatmodeler   
   - Repeatmasker       
   - MITE_hunter                                  
   - LTRharvest                                     
   - Uclust                                              
   - Biopython			      
   - Blast2

Download the full manual from the localizaTE manual release for further instructions
