
==========
localizaTE  
==========
 
de novo and homology based Transposable Element annotation system

Prerequesites: 

   - Repeatmodeler (version 1-0-7)       
   - Repeatmasker (version 4-0-5)        
   - MITE_hunter                                  
   - LTRharvest                                     
   - Uclust                                              
   - Biopython			      
   - Blast2

Download the full localizaTE manual for further instructions
